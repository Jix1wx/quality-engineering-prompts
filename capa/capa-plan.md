# CAPA Plan Generation

Generate a Corrective and Preventive Action (CAPA) plan for:

**Nonconformance:**
{{nc_description}}

**Requirements:**
1. Immediate correction (containment)
2. Root cause analysis method to be used
3. Corrective action with owner and due date
4. Preventive action to prevent recurrence
5. Effectiveness verification criteria
6. 30/60/90 day follow-up schedule

**Format as a table with columns: Action, Owner, Due Date, Verification Method, Status**
