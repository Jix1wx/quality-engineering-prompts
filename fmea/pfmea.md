# PFMEA Generation

Generate a Process Failure Mode and Effects Analysis for:

**Process:**
{{process_description}}

**Requirements:**
- List process steps in sequence
- For each step: potential failure mode, effect, cause, current controls
- Rate: Severity (1-10), Occurrence (1-10), Detection (1-10)
- Calculate RPN (Risk Priority Number)
- Recommend actions for RPN > 100 items

**Format as a PFMEA table per AIAG/VDA FMEA Handbook.**
